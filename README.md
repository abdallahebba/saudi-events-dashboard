# 📊 Saudi Events Dashboard - Power BI

Interactive Power BI dashboard designed to monitor operational performance, events, reports, visits, and task progress through an intuitive and dynamic interface.

---

## 🚀 Dashboard Features

- 📌 KPI cards for tracking operational indicators and performance metrics
- 🗺️ Interactive Saudi Arabia map used as a dynamic filter across the dashboard
- 📊 Task status analysis with visual charts and performance distribution
- 📈 Compliance tracking using gauge charts and analytical visuals
- 🔎 Drill-through functionality for navigating into detailed activity information
- 🎛️ Interactive slicers and filters for better data exploration
- 📋 Embedded detailed panels inside the same page for smoother user experience
- 📍 Region-based analysis and operational monitoring
- 📑 Reports tracking and visits overview
- 🎨 Custom UI/UX layout designed for easier navigation and readability
